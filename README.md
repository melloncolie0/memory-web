# Memory Web

Интерактивный граф личных заметок — визуализация воспоминаний и мыслей в виде связанных узлов.

## Возможности

- 🧠 **Граф заметок** — узлы представляют заметки, линии — связи между ними
- 🎨 **Категории** — мультивыбор нескольких категорий одновременно
- 🔗 **Drag-to-connect** — долгое нажатие (500ms) активирует режим соединения
- ✋ **Drag & drop** — свободное перемещение узлов
- 🗑️ **Удаление связей** — тап по ребру
- 🌑 **Dark mode**
- 🇷🇺 **Русский интерфейс**

## Технологии

- React 18 (CDN)
- Tailwind CSS v3
- react-force-graph-2d
- Babel (in-browser)

## Запуск

Открыть `memory-web.html` в браузере.
